### HTTP APIs

|-------------|----------|----------------------------|

## How To Start

1. Pastikan port 5672, 27017, dan 8080 tersedia
2. Pada root folder, jalankan `docker compose up`
3. Panggil http://localhost:8080 untuk memeriksa apakah service hidup
